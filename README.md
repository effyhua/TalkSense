# 💬 TalkSense 言感 - 多智能体社交助手

TalkSense 言感 是一款多智能体社交助手，模拟一群性格各异的靠谱好友群聊，为你提供社交支持。

## ✨ 功能特点

### 🔍 深层意图解析
每个智能体会自动分析输入文本的深层意图、情绪状态和潜台词，然后根据自己的风格给出回复建议。

### 👥 智囊团回复体系
摒弃单一答案，通过3个 AI 智能体从不同视角给出方案，以微信群聊形式呈现：

- **✨ 爽爽**：以自我为主的体验玩家，高能量Alpha。通过她的视角，关系不是一场考试，而是一场游戏。她只关注你爽不爽，不快乐就换。
- **💝 温荣**：无条件站队的情绪嘴替，你最忠诚的闺蜜/兄弟。无论你想做什么，她都先接住你的委屈，再帮你达成目的。
- **🧊 张凉**：犀利直球手，冷酷的"法医"。不负责哄你开心，只负责解剖事实。帮你计算投入产出比，强制止损。

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key（可选）

如果你有 Google Gemini API Key，可以创建 `.env` 文件：

```bash
cp .env.example .env
# 然后编辑 .env 文件，填入你的 GEMINI_API_KEY
```

获取 Gemini API Key：访问 [Google AI Studio](https://makersuite.google.com/app/apikey)

如果不配置 API Key，应用将使用演示模式（基于规则的回复）。

### 3. 运行应用

```bash
streamlit run app.py
```

应用将在浏览器中自动打开，默认地址：`http://localhost:8501`

## 📖 使用说明

1. 在输入框中输入你的社交场景或对话内容
2. 点击"发送"按钮
3. 3个智能体会自动分析你的情况，并从各自的角度给出回复建议
4. 所有回复以微信群聊的形式呈现，每个智能体像群成员一样发言

## 🎨 界面特点

- 微信群聊风格的界面设计

## ⚙️ 配置说明

### 使用 Gemini API（推荐）

配置 API Key 后，系统会使用 Gemini 3 Flash Preview 进行更精准的回复生成。每个问题都会让3个智能体都回答，提供不同视角的建议。

**注意**：Gemini 免费层每日有20次请求限制。如果达到限制，系统会显示友好提示，并建议使用演示模式。

### 演示模式

如果不配置 API Key 或达到配额限制，系统会使用基于规则的简单回复，仍然可以体验完整功能。每个问题也会让3个智能体都回答。

### API 错误处理

系统内置了智能错误处理：
- **配额限制**：显示友好提示，不进行无效重试
- **临时错误**（503、过载等）：自动重试，最多3次，使用指数退避策略
- **其他错误**：自动回退到演示模式
